RIA.com
============
Мы начали создавать новые персонализированные API, которые помогут Вам управлять всеми Вашими обьявлениями на RIA.com

При этом, нет необходимости открывать много вкладок, теперь все API доступны в одной админ-панели.
Зарегистрируйтесь или авторизируйтесь на _`Developers.RIA.com <https://developers.ria.com/>`_, чтобы использовать персонализированные API. По умолчанию все
права на доступ к сервисам не активны, что бы активировать нужный Вам сервис перейдите во вкладку _`"Мои разрешения" <https://developers.ria.com/account/credentials>`_

Все методы и данные описаные с помощью стандарта DeFacto swagger 2.0 _`здесь <http://swagger.ria.com/ui/?api=ria/basket>`_
